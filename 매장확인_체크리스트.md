# 코스트코 매장 전용 인기템 — 현장 확인 체크리스트

> 매장 베이커리·델리는 **코스트코 온라인 카탈로그에 없음**(API 조회로 확인).
> 따라서 아래 목록은 커뮤니티·블로그 조사 기반 **후보**이며, **현장에서 실물 확인이 필요**합니다.
> 확인할 항목: ① 실제 판매 여부 ② 정확한 상품명 ③ 가격 ④ 용량/수량 ⑤ 유통기한 ⑥ 1인 구매수량 제한

---

## A. 배송 적합 — 우선 추천 (상온 베이커리)

포장이 쉽고 택배 사고 위험이 낮아 대행 상품으로 가장 적합합니다.

| 후보 상품 | 조사 내용 | 확인란 |
|---|---|---|
| **크루아상** | 베이커리 대표 스테디셀러. 대용량 1만원 이하 언급 | 판매 ☐ / 가격 ____ / 용량 ____ |
| **머핀 (대용량)** | 블루베리·초코칩·바나나넛 중 2종 선택 가능 형태로 언급 | 판매 ☐ / 가격 ____ / 용량 ____ |
| **디너롤** | 버터 풍부한 프렌치 스타일 번. 샌드위치용 인기 | 판매 ☐ / 가격 ____ / 용량 ____ |
| **커클랜드 베이글** | 2팩 1만원 초반대 언급 | 판매 ☐ / 가격 ____ / 용량 ____ |
| **파운드케이크 / 크룽지 / 호두파이류** | 베이커리 진열 품목으로 자주 언급 | 판매 ☐ / 가격 ____ / 용량 ____ |

⚠️ **주의**: 매장 베이커리는 방부제를 최소화해 **유통기한이 2~3일**로 짧습니다.
→ 수요일 발송 → 목·금 도착이면 **남은 기한이 하루 남짓**일 수 있음.
→ 상품 설명에 "수령 후 빠른 섭취 권장 / 냉동 보관 가능" 안내 필요.

## B. 배송 신중 검토 — 냉장 조리식품

인기는 높지만 **식품안전·클레임 위험**이 큽니다. 첫 회차에는 제외하거나 소량 테스트를 권장합니다.

| 후보 상품 | 조사 내용 | 배송 리스크 |
|---|---|---|
| **로티세리 치킨** | 6천원대, 델리 대표 메뉴 | 조리 완료 온장식품 → 상온 이동 시 변질 위험 **높음** |
| **로티세리 계육다리** | 닭다리 5개 포장 형태 언급 | 동일 |
| **델리 피자 (홀)** | 델리 전용 주방 조리, 대형 | 부피 큼 + 냉장 유지 어려움 |
| **델리 샐러드·샌드위치류** | 대용량, 4인 가족 기준 | 신선도 민감, 유통기한 매우 짧음 |

💡 **판단 기준**: 보냉 포장으로 **다음날 도착**이 보장되는 지역(수도권)에 한정하거나,
아예 "매장 픽업 문의" 항목으로만 두는 것도 방법입니다.

## D. 매장에서 함께 해두면 좋은 것

- [ ] 상품 **사진 촬영** (현재 일러스트 → 실사진 교체용, 신뢰도 상승)
- [ ] 매장 카트·계산 영수증 사진 (사이트에 "실제 구매 인증"으로 활용)
- [ ] 베이커리 **재입고 시간** 확인 (인기품 오전 10~11시 진열 언급 있음)
- [ ] 1인당 **구매수량 제한** 있는 품목 메모 (사이트 max_qty 반영)
- [ ] 냉장·냉동 코너 인기템 추가 발굴 (연어·소불고기·만두 등)

## 수정 방법

확인한 내용을 알려주시면 `data/store_only.json`을 수정합니다.
이미지는 `images/store/` 폴더의 같은 이름 파일로 교체하면 자동 반영됩니다.
