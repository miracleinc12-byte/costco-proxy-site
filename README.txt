코스트코 대행 웹사이트 — 설정 가이드
══════════════════════════════════════════════════

[ 파일 구조 ]
costco-proxy-site/
├── index.html          ← 메인 랜딩 페이지
├── products.html       ← 이번 주 상품 목록
├── order.html          ← 주문 / 결제 페이지
├── complete.html       ← 주문 완료 페이지
├── css/style.css       ← 전체 스타일
└── data/products.json  ← ★ 매주 Claude가 업데이트

────────────────────────────────────────────────
STEP 1. 이메일 주소 설정 (주문 알림 수신)
────────────────────────────────────────────────
order.html 파일을 열어 아래 부분을 찾아주세요:

  <form action="https://formsubmit.co/YOUR_EMAIL@example.com"

YOUR_EMAIL@example.com 을 본인 이메일로 교체합니다.
예) https://formsubmit.co/miracleinc12@gmail.com

처음 주문이 들어오면 formsubmit.co에서 이메일 인증 요청이 옵니다.
인증 링크를 클릭하면 이후 자동으로 주문 내용이 이메일로 전달됩니다.

────────────────────────────────────────────────
STEP 2. 계좌번호 설정
────────────────────────────────────────────────
order.html 과 complete.html 에서 아래 내용을 본인 정보로 교체:

  3333-00-0000000   → 본인 계좌번호
  카카오뱅크        → 본인 은행
  홍길동            → 본인 이름

────────────────────────────────────────────────
STEP 3. 무료 호스팅 (GitHub Pages) — 권장
────────────────────────────────────────────────
1. github.com 에 무료 계정 가입
2. 새 Repository 생성 (이름: costco-proxy-site)
3. 이 폴더 안의 모든 파일을 업로드
4. Settings → Pages → Source: main branch / root
5. 몇 분 뒤 https://사용자명.github.io/costco-proxy-site 로 접속 가능

또는 netlify.com 에서 폴더째 드래그앤드롭으로 즉시 배포 가능합니다.

────────────────────────────────────────────────
STEP 4. 매주 상품 업데이트 방법 (Claude가 진행)
────────────────────────────────────────────────
매주 월요일:
1. Claude에게 "이번 주 코스트코 할인 상품 업데이트해줘" 요청
2. Claude가 data/products.json 파일을 새로운 내용으로 작성
3. GitHub에서 해당 파일만 교체 업로드 (1분 작업)
4. 사이트에 자동 반영 완료

────────────────────────────────────────────────
STEP 5. 도메인 연결 (선택 — 나중에)
────────────────────────────────────────────────
gabia.com 또는 cafe24에서 도메인 구매 후 GitHub Pages에 연결 가능
예) www.코스트코대행.com

══════════════════════════════════════════════════
문의: Claude에게 추가 기능 요청 가능
- 카드결제 추가 (토스페이먼츠 연동)
- 주문 관리 페이지 (어드민)
- 카카오 알림톡 자동 발송
